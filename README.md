# Blog
spring boot+vue前后端分离博客项目（前端博客+后台管理+后台服务）
> 简介：使用spring boot+vue构建的前后端分离博客项目，有前端界面+管理界面+后台服务的整套系统。

技术栈：

    SpringBoot 2.x 后台基本框架
    Vue + ElementUI
    ElasticSearch 
    Shiro +Redis +JWT 
    Mybaits-Plus 
    
> Github地址:https://github.com/15626862046/Blog
